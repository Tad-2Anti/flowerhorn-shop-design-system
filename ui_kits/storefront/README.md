# FlowerHorn Shop — Storefront UI Kit

## Overview
Full interactive prototype for the FlowerHorn Shop e-commerce website.
Built with React + Babel, DM Sans, and Lucide icons.

## Pages / Screens
1. **Homepage** — Hero, categories, featured fish, trust bar, about, footer
2. **Shop / Catalog** — Filter sidebar, sortable product grid, chips
3. **Product Detail** — Specs, care guide, add to cart, related products
4. **Cart Drawer** — Slide-in cart with quantity controls and checkout

## Components
- `index.html` — Entry point; imports all component scripts
- `components/Header.jsx` — Sticky navigation with cart badge
- `components/HomePage.jsx` — Full homepage layout
- `components/ShopPage.jsx` — Catalog with filters
- `components/ProductPage.jsx` — Product detail view
- `components/CartDrawer.jsx` — Cart slide-in overlay
- `components/Footer.jsx` — Site footer

## Design Width
Desktop: 1280px reference; fully responsive down to 375px mobile.
